# Product Landing Page

プロダクト紹介用のランディングページ

---

## 🎯 機能

- ✅ ヒーローセクション（キャッチコピー＋CTA）
- ✅ 機能紹介セクション（3つの特徴）
- ✅ CTA（Call To Action）セクション
- ✅ フルレスポンシブデザイン
- ✅ スムーズスクロール

---

## 🛠️ 技術スタック

- **HTML5**: セマンティックHTML
- **Tailwind CSS**: CDN経由で利用
- **Vanilla JavaScript**: スムーズスクロール

---

## 📦 ファイル構成

```
landing-page/
├── README.md          # このファイル
├── SPEC.md            # 仕様書
├── src/
│   ├── index.html     # メインHTML
│   └── script.js      # JavaScript（最小限）
└── .claude/           # Claude Code設定
```

---

## 🚀 使い方

### ローカルで実行

1. `src/index.html` をブラウザで開く
2. または、VS Codeの Live Server を使用

### Vercelにデプロイ

1. GitHubにプッシュ
2. Vercelで Import
3. Deploy

---

## 📝 学べること

### 1. Tailwind CSS
- ユーティリティファーストCSS
- レスポンシブデザイン（sm:, md:, lg:）
- グラデーション、シャドウ

### 2. セマンティックHTML
- `<header>`, `<main>`, `<section>`, `<footer>`
- アクセシビリティ

### 3. レイアウト
- Flexbox / Grid
- コンテナ幅の管理
- スペーシング
