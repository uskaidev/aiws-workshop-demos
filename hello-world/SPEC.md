# Hello World プロジェクト仕様書

このファイルは **Kiro式仕様書駆動開発** のエッセンスを1ファイルで体験できるように構成されています。

**Kiro式の思想**:

- **要件定義**: 何を作るか（目的・機能）
- **設計**: どう作るか（デザイン・技術仕様）
- **実装手順**: どの順番で作るか（ステップ）

## Part 1: 要件定義 - 何を作るか

- Claude Codeを使ったSpec駆動開発を体験する
- HTMLとCSSの基礎を理解する
- Git/GitHub/Vercelのデプロイフローを学ぶ

### 受け入れ基準

- [ ] HTMLが正しく表示される

- [ ] 絵文字がアニメーションする
- [ ] スマホでも正しく表示される（レスポンシブ）
- [ ] Vercelにデプロイできる

## Part 2: 設計 - どう作るか

### 🎨 デザイン要件

### 全体的な印象
- **シンプル**: 余計な要素は一切なし
- **モダン**: グラデーション背景とグラスモーフィズム
- **魅力的**: 見た人が「おっ」と思うデザイン

### カラースキーム
- 背景：紫系のグラデーション（例：`#667eea` → `#764ba2`）
- テキスト：白色
- カード背景：半透明の白（グラスモーフィズム）

### レイアウト
- 画面中央にコンテンツを配置
- フレックスボックスで縦横中央揃え
- レスポンシブ対応（PC・タブレット・スマホ）

---

## 📄 HTMLファイル仕様 (index.html)

### 必須要素

#### 1. DOCTYPE宣言
```html
<!DOCTYPE html>
```

#### 2. HTMLタグ
- 言語属性：`lang="ja"`

#### 3. headセクション
- 文字コード：UTF-8
- ビューポート設定：レスポンシブ対応
- タイトル：「Hello World」
- 説明：meta description（任意の内容）
- スタイルシートリンク：`style.css`

#### 4. bodyセクション

**コンテナ要素**:
- クラス名：`container`
- 中央配置されたカードのようなデザイン

**メインコンテンツ**（コンテナ内に以下を含む）:
1. **見出し（h1）**
   - テキスト：「Hello World」
   - 大きく太字で表示

2. **サブタイトル（p）**
   - クラス名：`subtitle`
   - テキスト：「ようこそ、Claude Code ワークショップへ」
   - 見出しより小さく、軽めのフォント

3. **情報セクション（div）**
   - クラス名：`info`
   - 上部にボーダーライン
   - 以下の2つのパラグラフを含む：
     - 「このページは **Vercel** にデプロイされています」
     - 絵文字「🎉」のみ（大きく表示、アニメーション付き）

### セマンティックHTML
- 適切なタグを使用（div、section、header等）
- クラス名は意味のあるものを使用

## 🎨 CSSファイル仕様 (style.css)

### 1. CSS Reset（minimal）
```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

### 2. Body スタイル
- フォントファミリー：システムフォント（Apple、Windows対応）
  ```
  -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
  ```
- 高さ：`min-height: 100vh`（画面いっぱい）
- 表示：フレックスボックスで中央揃え
- 背景：グラデーション（135度、紫系）
  ```css
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  ```
- テキスト色：白
- パディング：`1rem`（スマホ対応）

### 3. Container スタイル
- テキスト揃え：中央
- 最大幅：`600px`
- 幅：`100%`
- パディング：`2rem`
- 背景：半透明の白
  ```css
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  ```
- ボーダー半径：`20px`（丸角）
- 影：柔らかいシャドウ
- ボーダー：半透明の白

### 4. Typography
- **h1**:
  - フォントサイズ：`3rem`
  - フォントウェイト：`700`（太字）
  - マージン下：`1rem`
  - レタースペーシング：`-0.02em`（少し詰める）

- **subtitle**:
  - フォントサイズ：`1.25rem`
  - フォントウェイト：`300`（軽め）
  - マージン下：`2rem`
  - 透明度：`0.9`

### 5. Info セクション
- マージン上：`2rem`
- パディング上：`2rem`
- ボーダー上：半透明の白のライン
- パラグラフ：マージン `0.5rem 0`、フォントサイズ `1rem`

### 6. Emoji スタイル
- クラス名：`emoji`
- フォントサイズ：`3rem`
- マージン上：`1rem`
- アニメーション：バウンス（上下に動く、2秒ループ）
  ```css
  @keyframes bounce {
      0%, 100% { transform: translateY(0); }
      50% { transform: translateY(-10px); }
  }
  ```

### 7. レスポンシブ対応

#### タブレット（768px以下）
- h1: フォントサイズ `2rem`
- subtitle: フォントサイズ `1rem`
- container: パディング `1.5rem`

#### スマホ（480px以下）
- h1: フォントサイズ `1.5rem`
- subtitle: フォントサイズ `0.9rem`
- emoji: フォントサイズ `2rem`
