# Simple Todo App

シンプルなTodoリスト管理アプリ

---

## 🎯 機能

- ✅ Todoの追加
- ✅ Todoの削除
- ✅ 完了/未完了の切り替え
- ✅ ローカルストレージに自動保存
- ✅ レスポンシブデザイン

---

## 🛠️ 技術スタック

- **HTML5**: 構造
- **CSS3**: スタイリング（グラデーション、アニメーション）
- **Vanilla JavaScript**: ロジック
- **localStorage**: データ永続化

---

## 📦 ファイル構成

```
todo-app/
├── README.md          # このファイル
├── SPEC.md            # 仕様書（Spec駆動開発の見本）
├── src/
│   ├── index.html     # メインHTML
│   ├── style.css      # スタイルシート
│   └── script.js      # JavaScript
└── .claude/           # Claude Code設定
```

---

## 🚀 使い方

### ローカルで実行

1. `src/index.html` をブラウザで開く
2. または、VS Codeの Live Server を使用

### Vercelにデプロイ

1. GitHubにプッシュ
2. Vercelで Import
3. Deploy

## 📝 学べること

### 1. DOM操作
- `createElement`, `appendChild`, `removeChild`
- イベントリスナー（`addEventListener`）

### 2. ローカルストレージ
- `localStorage.setItem()`
- `localStorage.getItem()`
- JSON.parse() / JSON.stringify()

### 3. 配列操作
- `push()`, `filter()`, `map()`
- イミュータブルな更新

### 4. CSS
- Flexbox レイアウト
- トランジション・アニメーション
- レスポンシブデザイン
